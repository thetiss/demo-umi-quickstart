# umi project

通过umi脚手架实现react+dva+umi+antd,本repo有counter,todolist,userdashboard功能，分别放进不同路由。

## Getting Started

Install dependencies,

```bash
$ yarn
```

Start the dev server,

```bash
$ yarn start
```
